# Contenido de Repositorio

El repositorio se organiza en módulospara el aprendizaje y operación en Cisco IOS. Cada directorio contiene documentación y ejemplos prácticos.

## 1. Fundamentos de IOS

- Comandos básicos para iniciar la administración de cualquier equipo Cisco:
  - Acceso a privilegios
  - Navegación en IOS
  - Guardar y cargar configuraciones
  - Gestión de contraseñas
  - Alias y configuración básica de consola

## 2. Configuración de Switches

- Incluye comandos estructurados para:
  - Configuración de interfaces
  - VLANs y VTP
  - Trunking 802.1Q
  - EtherChannel (LACP / PAgP)
  - STP y sus variantes
  - Port-Security

## 3. Configuración de Routers

- Documentación de configuraciones típicas:
  - Configuración de interfaces RIPv2, OSPF, EIGRP
  - Rutas estáticas
  - NAT/PAT
  - DHCP en routers
  - ACL estándar y extendida

## 4. Servicios de Red

- Configuración de:
  - DHCP
  - NTP
  - Syslog
  - SNMP
  - DNS estático

## 5. Seguridad en Cisco IOS

- Buenas prácticas y comandos para:
  - Hardening básico
  - Privilege levels
  - SSH vs Telnet

## 6. Troubleshooting

- Sección de comandos de diagnóstico:
  - show
  - debug
- verificaciones comunes
- patrones para diagnóstico rápido

## 7. Laboratorios

- Incluye prácticas listas para Packet Tracer:
- Ejercicios básicos
- Configuraciones guiadas
- Escenarios completos

# Requisitos

- Familiaridad básica con Cisco IOS.
- Packet Tracer, EVE-NG o GNS3 en caso de pruebas.
- Entorno Linux/Mac/Windows según herramientas de apoyo.
